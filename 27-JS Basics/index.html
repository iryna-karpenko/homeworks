Існує два способи використання JavaScript у файлі HTML. Перший передбачає вбудовування всього коду JavaScript у код HTML, а другий метод використовує окремий файл JavaScript, який викликається всередині елемента Script, тобто додається до тегів Script.

    Cтилі написання імен змінних:
  • camelCase
  • PascalCase
  • snake_case
  • kebab-case
